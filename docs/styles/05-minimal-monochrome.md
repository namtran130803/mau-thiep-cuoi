# 05 · Minimal Monochrome (Trắng đen tinh giản)

> Dùng kèm [`prompt-create-wedding-template.md`](../prompt-create-wedding-template.md).

| | |
|---|---|
| **Nhóm** | Hiện đại & Tối giản |
| **Cảm xúc** | High-fashion, kiến trúc, cá tính |
| **Phù hợp** | Cặp đôi yêu thiết kế tối giản |

## Mô tả thẩm mỹ

Đương đại tuyệt đối: chỉ trắng–đen–xám. Typography oversized, layout bất đối xứng. Loại bỏ “sến” cưới truyền thống.

## Palette

| Vai trò | Hex |
|---------|-----|
| Trắng | `#FFFFFF` |
| Đen | `#000000` |
| Xám | `#E5E5E5` |

## Typography

- *Inter*, *Space Mono*
- Tiêu đề cực lớn (oversized), body mảnh hoặc bold contrast

## Họa tiết

Grid lines mảnh, layout asymmetric, không hoa không texture hoa văn.

## Gợi ý UI

- **Envelope:** Trắng full, chữ đen lớn “OPEN”, không ornament
- **Hero:** Tên chiếm 60% viewport, ảnh đen trắng high contrast
- **Gallery:** Masonry đơn giản, không sepia

## Tránh blush-gold

Mọi màu hồng/vàng/đỏ pastel cưới.

## Paste vào `[PHONG_CÁCH]`

```
Minimal Monochrome: chỉ #FFFFFF, #000000, #E5E5E5. Font Inter/Space Mono, typography oversized, layout asymmetric, grid lines mảnh. Zero floral, zero gold/pink — anti blush-gold romantic.
```
