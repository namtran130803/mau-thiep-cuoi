## Palette

| Vai trò | Hex / CSS |
|---------|-----------|
| Gradient nền | `#E0C3FC` → `#8EC5FC` |
| Glass card | `rgba(255,255,255,0.35)` + blur |
| Chữ | `#1E1B4B` |
| Accent | `#6366F1` |

## Typography

*Inter*, *SF Pro*, *Kanit* — sans sạch, medium weight

## Họa tiết

`backdrop-blur`, border trắng mờ, shadow lớn mềm, bo góc `rounded-3xl`, glow nhẹ.

## Gợi ý UI

- **Envelope:** Gradient full, card glass giữa, nút mở trong suốt
- **Sections:** Mỗi block là glass card nổi trên nền gradient
- **Không** texture giấy cổ

## Paste vào `[PHONG_CÁCH]`

```
Modern Glassmorphism: gradient pastel #E0C3FC→#8EC5FC, card kính rgba(255,255,255,0.35) backdrop-blur, bo góc lớn, shadow mềm. Font Inter. Digital layered UI — KHÔNG giấy kem blush-gold, KHÔNG hoa rơi.
```
