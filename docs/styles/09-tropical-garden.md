# 09 · Tropical Garden (Khu vườn nhiệt đới)

> Dùng kèm [`prompt-create-wedding-template.md`](../prompt-create-wedding-template.md).

| | |
|---|---|
| **Nhóm** | Tự nhiên & Mộc mạc |
| **Cảm xúc** | Tươi mát, resort, bãi biển |
| **Phù hợp** | Cưới ngoài trời nhiệt đới |

## Palette

| Vai trò | Hex |
|---------|-----|
| Xanh đại ngàn | `#1A3A2B` |
| Xanh non | `#A3B18A` |
| Cát | `#E9D8A6` |
| Chữ | `#2D4A3E` |

## Typography

Script tự do (*Great Vibes* tiết chế) + *Montserrat* sans

## Họa tiết

Monstera, lá cọ, dương xỉ silhouette — **không hoa hồng**, không cánh hoa gold.

## Gợi ý UI

- **Envelope:** Xanh lá, lá silhouette góc
- **Hero:** Ảnh sáng tropical, chữ trắng trên overlay xanh
- **Decoration:** Lá rơi nhẹ màu xanh

## Paste vào `[PHONG_CÁCH]`

```
Tropical Garden: #1A3A2B, #A3B18A, #E9D8A6. Montserrat + script nhẹ. Monstera/palm/fern silhouette — TUYỆT ĐỐI KHÔNG hoa hồng blush-gold, KHÔNG wax seal vàng.
```
