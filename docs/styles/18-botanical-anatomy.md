## Palette

| Vai trò | Hex |
|---------|-----|
| Rêu tối | `#2F3E1B` |
| Mực sepia | `#3B2F2F` |
| Giấy cũ | `#E6DFD3` |
| Chữ | `#2A2118` |

## Typography

Spencerian-style script (hoặc *Cormorant* italic) + Roman serif cổ

## Họa tiết

Line-art mực mảnh: cấu tạo hoa, số thứ tự như plate khoa học, label Latin giả (optional).

## Gợi ý UI

- **Envelope:** Giấy vàng, một specimen sketch
- **Timeline:** Numbered plates
- **Không** watercolor blush loang
