# 03 · Old Money Elegance (Quý tộc tối giản)

> Dùng kèm [`prompt-create-wedding-template.md`](../prompt-create-wedding-template.md).

| | |
|---|---|
| **Nhóm** | Kinh điển & Sang trọng |
| **Cảm xúc** | Quiet luxury, giàu có thầm lặng |
| **Phù hợp** | Lễ trang trọng, club địa phương |

## Mô tả thẩm mỹ

“Old money”: không họa tiết phô trương — whitespace hoàn hảo, layout chừng mực. Monogram cặp đôi là điểm nhấn duy nhất.

## Palette

| Vai trò | Hex |
|---------|-----|
| Nền | `#F8F9FA` |
| Navy | `#0B1B3D` |
| Chữ | `#000000` |
| Viền | `#D1D5DB` |

## Typography

- *Garamond* hoặc *Times New Roman* (serif cổ điển)
- Kerning/letter-spacing **rất rộng**
- Sans phụ tối giản nếu cần

## Họa tiết

**Không** hoa văn phụ. Chỉ monogram chữ lồng tối giản (2 chữ cái) trên phong bì.

## Gợi ý UI

- **Envelope:** Trắng tinh, monogram navy center, một đường kẻ mảnh
- **Hero:** Tên cặp đôi nhỏ thanh lịch, nhiều khoảng trắng
- **Sections:** Border 1px xám, không gradient

## Tránh blush-gold

Vàng đồng, hồng, hoa, texture rườm rà.

## Paste vào `[PHONG_CÁCH]`

```
Old Money Elegance: trắng #F8F9FA, navy #0B1B3D, chữ đen #000000. Font Garamond/Times, letter-spacing rộng. Monogram tối giản, KHÔNG hoa văn, KHÔNG vàng đồng/hồng blush. Quiet luxury whitespace-first.
```
